Badges

By default, our badges are designed to be on the right side of elements with 8px of margin on the left. If you would like to use a badge on the left side of an element, add .badge--left and it will switch the margin to be on the right side.

Buttons

Default sized buttons should be the most common use case. Large should only be used for marketing and big announcements. Use the action button sparingly for UX purposes. When multiple buttons are next to each other, wrap them in class .button-group. The default is assuming the buttons will be on the right side of the page, therefore 8px of margin is added on the left of each button in the group. Use class .button-group--left for buttons on the left side of the page, and margin will be moved to the right sides.

Forms

Use <form> element or .form depending on your needs. Use .form-group to add margin to the bottom of each child of the form. .form-group will also put the label and input in our standard layout using css grid. At 980px, the form layout will change to a stacked view. Resize window to preview.

  <form>
  <div class="form-group">
    <label>Text Input Label</label>
    <input type="text" placeholder="Text Input Placeholder">
  </div>

  <div class="form-group">
    <label>Number Input Label</label>
    <input type="number" placeholder="Number Input Placeholder">
    <div class="form-group__help-text">This is helper text for a form input</div>
  </div>

  <div class="form-group">
    <label>Select Label</label>
    <select>
      <option value="option_1">Option 1</option>
      <option value="option_2">Option 2</option>
      <option value="option_3">Option 3</option>
      <option value="option_4">Option 4</option>
    </select>
  </div>
</form>
